# Security Policy

## Supported Versions

Only the latest version of *Cut the Rope: DX* is supported. All users should update to the [latest release](https://github.com/yell0wsuit/cuttherope-dx/releases/latest) to receive security patches and bug fixes. Older versions will not receive fixes.

## Scope

This policy covers the *Cut the Rope: DX* repository only. Vulnerabilities in upstream dependencies (such as MonoGame) are out of scope—please report those to their respective projects. If an upstream fix is relevant to this project, it will be incorporated in a future update.

## Reporting a Vulnerability

> [!CAUTION]
> Do not open a public issue to report security vulnerabilities.

Please report vulnerabilities privately using GitHub's security disclosure feature:

🔒 [Report via GitHub Security Advisories](https://github.com/yell0wsuit/cuttherope-dx/security/advisories)

We will respond within 48 hours.

## Disclosure Process

Once a vulnerability is confirmed:

1. A fix will be developed and tested privately.
2. A new release will be published with the fix included.

If the issue originates from an upstream dependency, we will track the upstream fix and incorporate it into the next update. The advisory will be published once that update is released.
